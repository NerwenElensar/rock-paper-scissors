# Rock, Paper, Scissors 🪨 🗒️ ✂️

A classic game. This game was created in two iterations.

## First iteration

A basic version without GUI. The player plays against the computer and inserts his weapon of choice into a prompt. The result of a round will be shown in the console. (Open Dev Tools and choose console). A game consists of 5 rounds. After the the game the winner will  be announced in the console.

### Learning outcome - first version(console only)** 💻

**HTML**

- How to embed a JS file into an html file

**Javascript**

- Variables
- Functions
- `console.log()`, `alert()` and `prompt()`
- for-loop

## Second iteration

A version with a simple User Interface. The user can choose their weapon with buttons. After every round the score is updated. After 5 rounds the game ends and the winner is announced. 

### Learning outcome - second version(with UI)** 💻

**CSS**
- Style a simple UI for the Rock, Paper, Scissors game without a given design

**Javascript**
- Learning about the Document Object Model
  - Accessing, manipulating and transforming the DOM
  - Using Eventlisteners
- Transferring the previously implemented logic to a logic with UI
