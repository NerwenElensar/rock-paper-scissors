# Rock, Paper, Scissors 🪨 🗒️ ✂️

A classic game. This is a basic version without GUI. The player plays against the computer and inserts his weapon of choice into a prompt. The result of a round will be shown in the console. (Open Dev Tools and choose console). A game consists of 5 rounds. After the the game the winner will  be announced in the console.

## Learning outcome 💻

#### HTML

- How to embed a JS file into an html file

#### Javascript

- Variables
- Functions
- `console.log()`, `alert()` and `prompt()`
- for-loop
